### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/we-meow-times.git
cd we-meow-times
```

2. Install dependencies:

```bash
bun install
# or with npm
npm install
```

3. Run the development server:

```bash
bun dev
# or with npm
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
